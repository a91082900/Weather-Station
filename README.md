# Weather-Station
A simple weather station using ESP8266.
## Components
  * ESP8266
  * DHT11 (Temperature - Humidity Sensor)
  * ML8511 (UV Sensor)

## Arduino Libraries
  * DHTesp (https://github.com/beegee-tokyo/DHTesp)
  * ESP8266 Arduino Core (https://github.com/esp8266/Arduino)

## Back-end Requirements
  * PHP
  * MySQL
